# Delta-V

Chicken Soup for the large data visualization's soul.

This framework simplifies some VERY difficult issues to address in handling graphics for
`large dynamic` visualizations.

## Installing

```sh
npm install -DE deltav
```

OR

```sh
yarn deltav
```
